# javascript-calorie-tracker-project

This is a vanilla javascript based project to perform CRUD operations on a calories tracked for food.
What you will learn:
1) Materialize
2) HTML
3) CSS
4) Javascript
5) DOM Manipulation
6) Module based javascript pattern
7) Local storage handling
